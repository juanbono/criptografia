# [criptografia]() 
[![Build Status](https://travis-ci.org/juanbono/criptografia.svg?branch=master)](https://travis-ci.org/juanbono/criptografia)

##### Useful commands:

``` sh
# Build the project.
stack build

# Run the test suite.
stack test

# Run the benchmarks.
stack bench

# Generate documentation.
stack haddock
```
